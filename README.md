# VFS-CSVPreviewer
「**VFS CSV-Previewer**」の取扱説明書を作成いたしました．
本ツールは，大規模なCSVファイルをブラウザ上で高速に閲覧・検索し，必要に応じてExcel形式へ変換するためのプロフェッショナル向けプレビューアです．

---

## VFS CSV-Previewer 取扱説明書

### 1．概要

**VFS CSV-Previewer** は，Vanilla Field Softhouseが提供する，メモリ消費を抑えた高速CSV閲覧ツールです．ブラウザのローカルリソースのみを使用するため，外部サーバーへデータを送信することなく，安全かつ機密性を保ったまま大容量データの確認が可能です．

### 2．各部名称と機能

#### 2.1 ヘッダー（Header）

* **VFS Logo & Brand**: ブランドアイデンティティを表示します．
* **THEMEボタン**: ダークモードとライトモードを瞬時に切り替えます．作業環境の明るさに合わせて調整してください．

#### 2.2 ファイル操作エリア（Upload Section）

* **ファイル選択**: 複数のCSVファイルを一括で読み込むことができます．読み込まれたファイルは下部にタブとして並びます．

#### 2.3 グローバル検索（Global Search）

* **全件スキャン**: 読み込まれている「すべてのファイル」を対象に，特定の文字列を検索します．
* **検索結果**: ヒットしたファイル名と行番号が表示されます．結果をクリックすることで，該当するファイルの該当箇所へ即座にジャンプします．

#### 2.4 ナビゲーション・情報パネル

* **GOTO / ページ操作**:
* 表示開始行を指定して「GOTO」をクリックすることで，任意の位置へ移動します．
* 「NEXT」ボタンで50行ずつ読み進めることができます．


* **統計情報**: ファイルの総レコード数およびファイルサイズをリアルタイムに表示します．
* **EXCEL EXPORT**: 現在表示しているタブの全データを，`.xlsx` 形式に変換してダウンロードします．

### 3．テーブル操作（Table View）

#### 3.1 データの閲覧

* 一度に50行ずつ描画する「ステップ読み込み」を採用しているため，数万行のデータでも動作が重くなることはありません．

#### 3.2 カラム（列）の管理

* **列の非表示**: テーブルの見出し（TH）をクリックすると，その列を一時的に非表示にできます．
* **非表示リスト**: 非表示にした列は「HIDDEN COLUMNS」エリアにリストアップされます．リスト内の項目をクリックすることで，再度表示させることができます．

### 4．ユースケース

1. **巨大ログの調査**: 数百MBのCSVを開く際，表計算ソフトがフリーズする場合でも，本ツールならスムーズに特定行を特定できます．
2. **機密データの簡易チェック**: データをクラウドにアップロードしたくない環境でのデータ確認に最適です．
3. **簡易WBSの書き出し**: CSV形式で出力されたスケジュールデータを閲覧し，そのままExcel報告用に変換する際などに活用いただけます．
